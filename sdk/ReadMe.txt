====================QQ登录Java for Server SDK使用说明====================

====================当前版本信息====================
当前版本：V2.0.0.0

发布日期：2012-12-28

文件大小：2.22M

版本支持：支持JDK5.0及以上版本。


====================修改历史====================

====================文件夹结构说明====================
Sdk4J.jar SDK for java 所有的相关类文件
qqconnectconfig.properties 文件 需开发者自行修改的配置文件
sdk4j_doc SDK java_doc 所在目录
sdk4j_demo SDK 演示程序所在目录


====================类包结构说明====================
所有的api调用相关的类都封装在了com.qq.connect.api类包下
初始化SDK需要调用com.qq.connect.oauth.Oauth类的相关方法完成Oauth相关授权
详细的类包借口可直接参考javaDoc文档

====================使用方法====================
1.直接引入Sdk4J.jar 包至项目工程内
2.修改qqconnectconfig.properties 文件，在指定修改的地方填写自己app的相关信息和要获取的scope权限（前4行信息）注意不要试图修改api的请求地址，这里之所以暴露出来是为了方便兼容以后的变动。
3.将qqconnectconfig.properties文件放到自己的项目的context ClassLoader的可以加载的目录下，一般放在项目的src目录即可。
4.查看demo程序，结合api doc文档，进行接口的调用。
